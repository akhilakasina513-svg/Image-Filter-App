IMAGE FILTER APP

Description:
This project is a GUI-based Image Filter Application developed using Python, Tkinter, OpenCV, and Pillow.

Features:
1. Upload images from the computer.
2. Apply Grayscale filter.
3. Apply Blur filter.
4. Apply Edge Detection filter.
5. Preview the original and filtered images side by side.
6. Save the filtered image.

Technologies Used:
- Python
- Tkinter
- OpenCV
- Pillow

How to Run:
1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   python filter.py
